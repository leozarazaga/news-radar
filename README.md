
# News Radar

"News Radar" is a Bootstrap application that delivers daily news, culture, sports, and film all in one place.

## Installation

To set up this project locally, follow these steps:

Clone the repository to your local machine using the following command:
```bash
git clone https://github.com/leozarazaga/news-radar.git
```
Navigate to the project directory:
```bash
cd news-radar
```

## Running the Project
To run the project, simply open the `index.html` file in your browser:

1. Locate the `index.html` file in the project directory.
2. Double-click the file, or right-click and select "Open with" and choose your preferred web browser.

## Technologies Used
- **Bootstrap:** For responsive design and layout.
- **Font Awesome:** For icons used throughout the application.
- **HTML & CSS:** For the structure and styling of the web pages.
